# MyItems Backend (Node + Express + SQLite)

CRUD completo de un único recurso `Item` con búsqueda, paginación y healthcheck de DB.

## Requisitos
- Node 18+

## Instalación
```bash
npm install
cp .env.example .env
```

## Arranque
- npm run dev   # desarrollo con nodemon
# o
- npm start     # producción
